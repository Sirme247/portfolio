MindMate

A comprehensive educational platform built with Flutter that connects students with tutors, provides course management, and facilitates collaborative learning through forums and messaging. The app serves as a one-stop solution for academic support and learning management.

Key Features

Authentication & User Management

Google Sign-In integration and Firebase authentication for secure user sessions and personalized profiles.

Course Management

Browse, enroll, and track progress across diverse courses with detailed modules and materials.

Tutor System

Find qualified tutors, schedule appointments, and manage personalized tutoring sessions.

Communication & Collaboration

Real-time messaging, discussion forums, and notifications for seamless academic interaction.

Network Monitoring

Automatic detection and handling of network connectivity with offline support.

Modern UI Design

Material Design 3 with dark/light theme support and responsive layouts.
